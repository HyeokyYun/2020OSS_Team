# mini project

### 주요 기능 
##### 1. 데이터 조회
##### 2. 데이터 추가
##### 3. 데이터 수정
##### 4. 데이터 삭제
##### 5. 데이터 파일 저장
##### 6. 데이터 파일 로드

##### 8. 점수(평균)로 검색 : 입력한 평균 이상의 평균을 가진 학생들을 출력
##### 9. 등수로 검색
##### 10. 등수에 따라 데이터 나열
